Example Require.js Multi-page
=============================

The source code of an example using Requirejs Multi-page.

## Requeriments

- Node.js
- NPM
- Bower
- Grunt

## Installation

Init example.

	npm install -g bower
	npm install -g grunt-cli
	git clone https://github.com/lagden/example-multipage-requirejs.git example
	cd example
	npm install -d
	
## Deploy

Generate deploy.

	grunt deploy
