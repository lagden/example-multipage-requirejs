Example Require.js Multi-page
=============================

Small example using Requirejs Multi-page.

## Requeriments

- Bower
- Grunt

## Installation

Init example.

```
git clone https://github.com/lagden/example-multipage-requirejs.git example
cd example
npm i -d
```

## Usage

```
npm start
```

## Deploy

```
npm run deploy
```
